先將菜單和折扣完整的json file 丟給AI讓他當背景知識
使用者每一次會主動給予當前所有有購買的品項(MenuItem)
期望根據items 裡面的內容找出最合適的折扣方案並套用(Strategy)折扣
也就是說，這一段AI呼叫完全可以直接放在策略模式(Context)裡面

在策略模式中，我們只需要給予items內容，讓他自己的Tool 自己決定選擇哪一種折扣方案，然後透過拆解該折扣對應的json  取得 stragegy Name
這樣我們就可以再透過反射拿到我們的折扣演算法

最後就可以呼叫並執行折扣演算法完成整件事，那又因為整體的過程都是call by reference (傳入items 去影響) 所以整件事交給其他tool 呼叫策略在進行事完全沒問題的

PS:這樣的話我們form/order 就不需要自己傳入下拉選單了，因為可以由AI判斷並取回
